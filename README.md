# Learning Progress Tracker

## Description
**Learning Progress Tracker** is a console-based Java application for managing students and tracking their learning progress. The program allows adding students, registering their achievements in various courses, and generating statistics on course popularity and difficulty.

## Features
- Add and manage students
- Register course grades
- Display overall statistics
- Determine course popularity and difficulty
- Generate reports
- Flexible interaction via console

## Installation and Execution
### Requirements:
- Java 17+

### Running Instructions:
1. Clone the repository:
   ```sh
   git clone https://github.com/GovernorofMarbas/Learning_Progress_Tracker_Final_Project.git
   ```
2. Navigate to the project folder:
   ```sh
   cd Learning_Progress_Tracker_Final_Project
   ```
3. Compile the project:
   ```sh
   javac -d out -sourcepath src src/Main.java
   ```
4. Run the application:
   ```sh
   java -cp out Main
   ```

## Usage
After launching the program, follow the instructions in the console. Available commands:
- `add students` — Add a new student
- `list` — Show all registered students
- `add points` — Add course points
- `statistics` — Display statistics
- `notify` — Notify students about their progress
- `exit` — Exit the program
